# Workspace for PostgreSQL with preconfigured PGAdmin

## Start or stop containers

### Windows

```
bash run.sh
```

### Linux

```
/bin/sh run.sh
```

## PgAdmin

Go to http://localhost:8071

## PostgreSQL

- Host: localhost
- Dockerhost: host.docker.internal
- Port: 5438
- User: admin
- Password: pass
